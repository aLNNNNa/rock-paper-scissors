# 石头剪刀布 · 人机对战

一个简单的命令行版石头剪刀布游戏，你和电脑对战，系统自动记录比分并判定最终胜负。

## 功能特性

- 玩家输入 1（石头）、2（剪刀）、3（布）进行游戏
- 电脑随机出拳
- 实时显示本轮结果和当前比分
- 可随时选择 4 退出，并显示最终比分和总结语
- 支持输入错误处理

## 如何运行

确保已安装 Python 3.6+，在终端执行：

```bash
python game.py

=============石头剪刀布=============
        1.石头
        2.剪刀
        3.布
        4.退出并查看比分    
===================================
请选择你的操作（1-4）：1
电脑的操作为：2
恭喜你赢了！
