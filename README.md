# 石头剪刀布 · 人机对战

一个简单的命令行版石头剪刀布游戏，你和电脑对战，系统自动记录比分并判定最终胜负。

## ✨ 功能特性

- 玩家输入 1（石头）、2（剪刀）、3（布）进行游戏
- 电脑随机出拳
- 实时显示本轮结果（赢/输/平）和当前比分
- 可随时选择 4 退出，并显示最终比分和总结语
- 支持输入错误处理（非法数字、字母等）

## 🎮 如何运行

1. 确保你的电脑已安装 **Python 3.6+**
2. 克隆或下载本仓库到本地
3. 打开终端，进入项目目录
4. 执行以下命令：
